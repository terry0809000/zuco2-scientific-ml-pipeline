# Security and Data Policy

This repository contains code and documentation only. It must not include raw ZuCo `.mat` files, private local data, generated word-level tables, model artifacts, or credentials.

## Reporting Issues

Open a GitHub issue for code, reproducibility, dependency, or documentation problems. Do not attach raw datasets, private access tokens, or restricted files.

## Data Handling

Use local storage or private Google Drive storage for ZuCo files. The `.gitignore` file excludes MATLAB/HDF5 data files and generated outputs by default.
